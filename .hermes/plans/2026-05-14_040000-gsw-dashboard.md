# GSW Dashboard — IGA Competitive Intelligence

> **For Hermes:** Execute this plan task-by-task. Use `subagent-driven-development` if multi-step delegation is appropriate.

**Goal:** Build a beautiful, data-dynamic competitive intelligence dashboard for IGA Vashon that shows pricing position vs Thriftway with actionable insights.

**Architecture:** Server Components fetch from existing API routes at build/request time. Client components handle interactivity (tabs, search, tooltips). Recharts for all visualizations. shadcn/ui for UI primitives. Dark mode from existing layout.

**Tech Stack:** Next.js 16 (App Router), React 19, Recharts 3.8.1, shadcn/ui, Tailwind 4, Turso (libsql)

**IGA-centric design language:** Green = IGA advantage (cheaper, winning), Red = IGA disadvantage (more expensive, losing), Amber = caution. Dashboard answers one question: "How does IGA beat Thriftway?"

**Live data only** — no mock data, no hardcoded values. All API routes are built and return real Turso data.

Actually — we need a new API route for this. Let me add a quick one.

**Step 1: Create price distribution API endpoint**

```tsx
// frontend/src/app/api/prices/distribution/route.ts
import { NextResponse } from "next/server";
import turso from "@/db";

export async function GET() {
  const result = await turso.execute(`
    SELECT
      ROUND(p.price - pt_comp.price, 2) as gap
    FROM products p
    JOIN product_matches pm ON p.id = pm.iga_product_id
    JOIN products pt_comp ON pm.thriftway_product_id = pt_comp.id
    WHERE p.store_id = 'iga-vashon'
      AND p.price IS NOT NULL
      AND pt_comp.price IS NOT NULL
  `);

  const gaps = result.rows.map((r: any) => Number(r.gap));
  return NextResponse.json({ gaps });
}
```

**Step 2: Create visualization component**

```tsx
// frontend/src/components/dashboard/price-position-chart.tsx
"use client";
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";
import { Skeleton } from "@/components/ui/skeleton";
import { BarChart, Bar, XAxis, YAxis, CartesianGrid, Tooltip, ResponsiveContainer, Cell } from "recharts";
import useSWR from "swr";

const fetcher = (url: string) => fetch(url).then(r => r.json());

function binGaps(gaps: number[]) {
  const bins: Record<string, { range: string; count: number; color: string }> = {};
  const bucketSize = 0.50;
  
  for (const gap of gaps) {
    const bucketLow = Math.floor(gap / bucketSize) * bucketSize;
    const key = `${bucketLow.toFixed(2)}`;
    if (!bins[key]) {
      const color = bucketLow < -0.25 ? "#10b981" : bucketLow > 0.25 ? "#ef4444" : "#f59e0b";
      bins[key] = {
        range: `$${bucketLow.toFixed(2)} to $${(bucketLow + bucketSize).toFixed(2)}`,
        count: 0,
        color,
      };
    }
    bins[key].count++;
  }
  return Object.values(bins).sort((a, b) => parseFloat(a.range) - parseFloat(b.range));
}

export function PricePositionChart() {
  const { data, isLoading } = useSWR("/api/prices/distribution", fetcher);

  const chartData = data?.gaps ? binGaps(data.gaps) : [];

  if (isLoading) {
    return (
      <Card>
        <CardHeader><CardTitle>Price Position Distribution</CardTitle></CardHeader>
        <CardContent><Skeleton className="h-64 w-full" /></CardContent>
      </Card>
    );
  }

  return (
    <Card>
      <CardHeader>
        <CardTitle>Price Position vs Thriftway</CardTitle>
        <p className="text-sm text-zinc-500">Negative = IGA cheaper. Positive = IGA more expensive.</p>
      </CardHeader>
      <CardContent>
        <ResponsiveContainer width="100%" height={300}>
          <BarChart data={chartData}>
            <CartesianGrid strokeDasharray="3 3" className="stroke-zinc-200 dark:stroke-zinc-800" />
            <XAxis dataKey="range" tick={{ fontSize: 10 }} angle={-30} textAnchor="end" height={60} />
            <YAxis tick={{ fontSize: 12 }} />
            <Tooltip />
            <Bar dataKey="count" radius={[4, 4, 0, 0]}>
              {chartData.map((entry, index) => (
                <Cell key={index} fill={entry.color} />
              ))}
            </Bar>
          </BarChart>
        </ResponsiveContainer>
        <div className="flex items-center gap-4 mt-3 text-xs text-zinc-500 justify-center">
          <span className="flex items-center gap-1"><span className="w-3 h-3 rounded bg-emerald-500" /> IGA cheaper</span>
          <span className="flex items-center gap-1"><span className="w-3 h-3 rounded bg-amber-500" /> Near parity</span>
          <span className="flex items-center gap-1"><span className="w-3 h-3 rounded bg-red-500" /> IGA more expensive</span>
        </div>
      </CardContent>
    </Card>
  );
}
```

// frontend/src/components/dashboard/staples-table.tsx
"use client";
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";
import { Table, TableBody, TableCell, TableHead, TableHeader, TableRow } from "@/components/ui/table";
import { Skeleton } from "@/components/ui/skeleton";
import { Badge } from "@/components/ui/badge";
import { ShoppingBasket } from "lucide-react";
import useSWR from "swr";

const fetcher = (url: string) => fetch(url).then(r => r.json());

export function StaplesTable() {
  const { data, isLoading } = useSWR("/api/staples", fetcher);

  if (isLoading) {
    return (
      <Card>
        <CardHeader><CardTitle>Staple Basket</CardTitle></CardHeader>
        <CardContent><Skeleton className="h-80 w-full" /></CardContent>
      </Card>
    );
  }

  const staples = (data || []).map((s: any) => {
    const igaPrice = Number(s.iga_price);
    const twPrice = Number(s.tw_price);
    const gap = igaPrice && twPrice ? twPrice - igaPrice : null;
    return { ...s, gap };
  });

  const igaTotal = staples.reduce((sum: number, s: any) => sum + (Number(s.iga_price) || 0), 0);
  const twTotal = staples.reduce((sum: number, s: any) => sum + (Number(s.tw_price) || 0), 0);

  return (
    <Card>
      <CardHeader>
        <div className="flex items-center justify-between">
          <CardTitle className="flex items-center gap-2">
            <ShoppingBasket className="h-5 w-5" />
            Staple Basket
          </CardTitle>
          <div className="flex items-center gap-3 text-sm">
            <span className="text-emerald-600 font-medium">IGA: ${igaTotal.toFixed(2)}</span>
            <span className="text-zinc-400">|</span>
            <span className="text-zinc-500">TW: ${twTotal.toFixed(2)}</span>
          </div>
        </div>
      </CardHeader>
      <CardContent>
        <Table>
          <TableHeader>
            <TableRow>
              <TableHead>Item</TableHead>
              <TableHead className="text-right">IGA</TableHead>
              <TableHead className="text-right">Thriftway</TableHead>
              <TableHead className="text-right w-20">Result</TableHead>
            </TableRow>
          </TableHeader>
          <TableBody>
            {staples.map((row: any, i: number) => (
              <TableRow key={i}>
                <TableCell className="font-medium">{row.iga_name || "—"}</TableCell>
                <TableCell className="text-right">{row.iga_display || "—"}</TableCell>
                <TableCell className="text-right">{row.tw_display || "—"}</TableCell>
                <TableCell className="text-right">
                  {row.gap == null ? (
                    <span className="text-zinc-400">—</span>
                  ) : row.gap > 0 ? (
                    <Badge className="bg-emerald-100 text-emerald-700 dark:bg-emerald-950 dark:text-emerald-400">
                      WIN +${row.gap.toFixed(2)}
                    </Badge>
                  ) : row.gap < 0 ? (
                    <Badge className="bg-red-100 text-red-700 dark:bg-red-950 dark:text-red-400">
                      -${Math.abs(row.gap).toFixed(2)}
                    </Badge>
                  ) : (
                    <Badge variant="outline">TIE</Badge>
                  )}
                </TableCell>
              </TableRow>
            ))}
          </TableBody>
        </Table>
      </CardContent>
    </Card>
  );
}

## Risks / Open questions

- **SWR not installed** — need to install `swr` package (`npm install swr`) or use native fetch in a server component pattern. If SWR doesn't work out of the box, fall back to `useEffect` + `useState` for client fetching.
- **No loading states for server components** — if we go server-component-first, we lose the skeleton loading. Either SWR client-side (skeleton UX) or React Suspense with server components.
- **Distribution API is new** — Task 4 adds an API route that computes a potentially large result set (5,215 rows). Test for performance; if slow, add LIMIT 1000 or compute bins server-side.
- **Dark mode** — Tailwind 4 + Geist font already supports it. All components use `dark:` variants.
